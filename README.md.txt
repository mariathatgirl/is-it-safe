# 🔐 Is it Safe?

**Is it Safe?** is a beginner-friendly cyber security project that analyzes a URL and shows whether it looks safe, suspicious, or risky based on simple security rules.

This project was created to practice **HTML, CSS, JavaScript, and basic cyber security concepts** in an ethical and educational way.

## 📌 About the Project

The goal of this project is to help users identify possible warning signs in links before clicking on them.

The tool checks basic URL characteristics, such as:

* HTTPS usage;
* suspicious words commonly used in phishing links;
* risky file extensions;
* URL shorteners;
* domains that may imitate famous brands;
* long or unusual domains;
* links that use IP addresses instead of domain names.

---

## ⚙️ Features

* Analyze a URL entered by the user;
* Generate a security score from 0 to 100;
* Show a risk level: low, medium, or high;
* Display clear warnings about suspicious signs;
* Detect risky file extensions such as `.exe`, `.zip`, `.apk`, and others;
* Detect popular URL shorteners such as `bit.ly`, `tinyurl.com`, and `cutt.ly`;
* Identify possible fake domains that imitate famous brands;
* Clear the analysis with a reset button;
* Responsive and modern dark cyber-themed interface.

## 🚨 Important Disclaimer

This project is for **educational purposes only**.

It uses simple rules to identify possible suspicious signs in a URL, but it does **not guarantee** that a website is completely safe or dangerous.

Always be careful before clicking links, downloading files, or entering personal information online.
