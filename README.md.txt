# Is it Safe?

Meu primeiro projeto de Cyber Security.

## Sobre o projeto

O Is it Safe? é um site educativo que permite ao usuário colar um link e receber uma análise inicial de segurança.

Ele verifica sinais simples, como:

- uso de HTTPS;
- domínio muito longo;
- domínio com números/IP;
- palavras comuns em golpes;
- arquivos perigosos como .exe, .zip e .apk;
- excesso de parâmetros na URL.

## Objetivo

O objetivo do projeto é aprender conceitos básicos de segurança digital, phishing e análise de links suspeitos de forma ética e segura.

## Tecnologias usadas

- HTML
- CSS
- JavaScript

## Aviso importante

Este projeto é educativo. Ele não invade sites, não coleta dados pessoais e não substitui ferramentas profissionais de segurança.

## Como usar

1. Abra o arquivo index.html no navegador.
2. Cole um link no campo de análise.
3. Clique em Analisar.
4. Veja a pontuação e os alertas.
